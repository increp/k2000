# k2000

VST3 synth plugin. Hybrid between **Novation Peak** (NCO oscillators + analog-modeled filter character + mod matrix) and **Kurzweil K2000 VAST** (user-configurable per-voice DSP graph).

Built with [JUCE](https://juce.com), in C++. Linux for local development; Windows builds produced by GitHub Actions for testing in Ableton 12.

## Status

**Pre-v1 / spec stage.** Design approved; implementation has not started yet. See [`docs/specs/2026-05-25-v1-skeleton-design.md`](docs/specs/2026-05-25-v1-skeleton-design.md) for what v1 will contain and what it intentionally won't.
